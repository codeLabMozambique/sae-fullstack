🛠️ Como Correr o Projecto

  1. Infraestrutura: Execute docker-compose up -d para subir os bancos de dados.
  2. Compilação: Execute mvn clean install -DskipTests na raiz.
  3. Execução: Inicie primeiro o sae-auth-service e depois o sae-api-gateway.

🔒 Segurança

Toda a segurança é centralizada no sae-common. Endpoints como /users/signup/** e /users/login estão abertos por padrão no Gateway.
